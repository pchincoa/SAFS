== Description ==
= Embed your P5JS sketches in posts and pages in a responsive way =
<br>
<strong>1. Create a new sub-directory in the /uploads/p5js/ directory (using FTP):
</strong><br><br>
<strong>2. Upload all files and directories (index.html, scetch.js, libraries, etc) of your P5JS sketch to the new WordPress /uploads/p5js/[sketch_folder] directory.</strong><br><br>
<strong>3. Add a short code to your post or page</strong>
<br><br>
<strong>Attributes</strong><br>
<strong>sketchfolder</strong>: the name of your P5JS sketch (REQUIRED)<br>
<strong>canvaswidth</strong>: the width of the sketch, should be the same as in your creattecanvas() statement (REQUIRED)<br>
<strong>canvasheight</strong>: the height of the sketch, should be the same as in your creattecanvas() statement (REQUIRED)<br>
<strong>bordersize</strong>: size of the border in pix (OPTIONAL, default: 0px)<br>
<strong>bordercolor</strong>: color of the border around the sketch (OPTIONAL, default: no border, only works when you also define a bordersize)<br>
